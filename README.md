# Netflix Content Analysis using Power BI

## Overview
This project analyzes the Netflix Movies and TV Shows dataset obtained from Kaggle. The data was cleaned and transformed using Power Query, followed by Exploratory Data Analysis (EDA) in Power BI.

## Tools Used
- Power BI
- Power Query

## Data Cleaning & Transformation
- Column profiling
- Handling missing values
- Encoding null values
- Imputing missing values
- Date cleaning and formatting
- Creating new columns
- Splitting and extracting data
- Extracting the first item
- Basic text analysis
- Filtering unnecessary rows and columns

## Exploratory Data Analysis
EDA was performed to understand the dataset, identify patterns, and prepare the data for analysis.

## Dataset
**Source:** Kaggle – Netflix Movies and TV Shows Dataset
